  • Each week begins : firstDay (check the code at line number : 797 below)
    The value must be a number that represents the day of the week.
    Sunday=0, Monday=1, Tuesday=2, etc.
  • Start time of a day : minTime (check at line number : 798 below)
    The default "00:00:00" means the start time will be at the very beginning of the day (midnight).
  • End time : maxTime (check at line number : 799 below)
    The default "24:00:00" means the end time will be at the very end of the day (midnight).
  • Slot Duration : slotDuration (check at line number : 800 below)
    The frequency for displaying time slots.
    default: '00:30:00' (30 minutes).
